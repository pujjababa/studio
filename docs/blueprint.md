# **App Name**: Hello World Display

## Core Features:

- Text Display: Display 'Hello, world!' on the screen.

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4) to symbolize clarity and approachability.
- Background color: Light blue (#E3F2FD), very lightly saturated, creating a calm and unobtrusive background.
- Accent color: A shade of purple (#673AB7), analogous to blue but with more saturation, will be used for interactive elements to stand out from the design.
- Font: 'Inter' (sans-serif) for a clean, modern, and readable interface.
